# v0.4 规划 - daisyUI 主题化重构

## 功能需求

- 集成 Tailwind CSS + daisyUI 组件库
- 支持 cupcake 和 dark 主题切换
- 重构现有组件，全面使用 Tailwind CSS 替代 CSS Modules
- 实现主题切换功能（可记忆用户选择）

## 实现步骤

1. **安装配置** → 安装 Tailwind CSS、daisyUI，配置主题
2. **主题切换** → 实现主题切换功能和状态管理
3. **组件改造** → 逐个改造现有组件（从 ChatInput 开始）
4. **样式清理** → 删除废弃的 CSS Modules 文件

## 技术要点

- Tailwind CSS utility-first 理念
- daisyUI 主题系统（data-theme 属性）
- Tailwind 与 daisyUI 的配合使用
- localStorage 主题记忆

## 测试验证

- 主题切换正常，无闪烁
- 所有组件样式正确显示
- cupcake 主题：粉嫩清新风格
- dark 主题：深色护眼风格

## 学习收获

- 掌握 Tailwind CSS 配置和使用
- 理解 daisyUI 组件库设计思想
- 学习现代 CSS 方案（utility-first）
- 主题系统设计实践
