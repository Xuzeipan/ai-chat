# v0.3 规划：Markdown 渲染

## 功能需求

1. **Markdown 渲染器组件** - 将 AI 返回的 Markdown 内容渲染为美观的 HTML
2. **代码高亮** - 支持代码块语法高亮（使用 highlight.js 或 react-syntax-highlighter）
3. **表格支持** - 支持 Markdown 表格渲染
4. **自定义样式** - 使用 CSS Modules 定义 Markdown 样式
5. **集成到 MessageBubble** - 在消息气泡中显示渲染后的 Markdown

## 实现步骤

```
步骤 1 → 步骤 2 → 步骤 3 → 步骤 4
```

1. **类型定义** - 定义 MarkdownRendererProps 接口
2. **MarkdownRenderer 组件** - 创建 Markdown 渲染组件
3. **集成到 MessageBubble** - 在消息气泡中使用 Markdown 渲染器
4. **样式优化** - 美化 Markdown 样式

## 技术要点

- **marked** - Markdown 解析库
- **highlight.js** - 代码高亮库
- **CSS Modules** - 组件样式隔离
- **DOMPurify** - XSS 防护（可选，但推荐）

## 测试验证

1. 测试基本 Markdown 语法（标题、列表、引用、代码块）
2. 测试代码块高亮效果
3. 测试表格渲染
4. 测试超链接和图片
5. 测试长文本滚动
6. 测试移动端响应式

## 学习收获

通过本版本你将学会：

- 如何在 React 中集成第三方库
- Markdown 解析和渲染原理
- 代码高亮的实现方式
- XSS 安全防护
- CSS Modules 高级用法
