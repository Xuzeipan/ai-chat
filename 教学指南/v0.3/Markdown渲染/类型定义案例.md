# 类型定义案例

## 基础实现

在 `src/types/index.ts` 中添加 Markdown 渲染相关的类型定义：

```typescript
// Markdown 渲染配置
export interface MarkdownRendererProps {
  content: string;        // Markdown 内容
  className?: string;     // 可选的 CSS 类名
  enableCodeHighlight?: boolean;  // 是否启用代码高亮（默认 true）
}
```

## 使用示例

```typescript
import type { MarkdownRendererProps } from "../types";

// 在 MessageBubble 中使用
interface MessageBubbleProps {
  message: Message;
  markdownProps?: Partial<MarkdownRendererProps>;  // 可选的渲染配置
}
```

## 原理解释

1. **`content`** - 要渲染的 Markdown 文本字符串
2. **`className`** - 允许外部传入额外的 CSS 类名，便于自定义样式
3. **`enableCodeHighlight`** - 控制是否对代码块进行语法高亮

## 你的任务

1. 打开 `src/types/index.ts`
2. 在文件末尾添加 `MarkdownRendererProps` 接口定义
3. 确保类型正确导出
